# 🩺 PerMed

**PerMed** is a research project developed as part of my Master's thesis in Artificial Intelligence.

The project explores the adaptation of Medical Large Language Models (LLMs) for the Persian language, aiming to improve medical question answering and multilingual healthcare applications.

## Research Focus

* Persian Medical Large Language Models
* Medical Question Answering
* Medical Text Translation
* Prompt Engineering
* Healthcare Artificial Intelligence

> 🚧 This repository is currently under active development. New features, documentation, and experimental results will be added progressively.

## System Architecture

PerMed is designed as a multilingual medical AI framework that extends the capabilities of medical large language models for Persian healthcare applications.

The framework consists of the following components:

1. Medical Knowledge Sources
2. Medical Large Language Model (Meditron)
3. Persian Language Adaptation
4. Prompt Engineering Layer
5. Medical Response Generation
6. Evaluation and Validation

The overall objective is to provide reliable Persian medical question answering while preserving the medical knowledge learned from English biomedical resources.
